# Dokumentasi & Literatur Proyek

Folder ini menyimpan seluruh dokumen pendukung, notulensi meeting, materi presentasi, dan kumpulan paper kajian pustaka (literature review) yang digunakan sebagai basis ilmiah dalam proyek **Mental Health Analysis**.

## 📌 Poin Utama
*   **Notulensi Lengkap**: Catatan detail hasil diskusi, arahan, dan notulensi pertemuan/meeting diarsipkan secara lengkap pada **Discord**. Silakan merujuk ke server Discord tim untuk history diskusi terupdate.
*   **Materi Presentasi**: PPT Pertemuan/Magang disimpan langsung di folder ini untuk review cepat.
*   **Kajian Pustaka**: Paper-paper rujukan dikelompokkan di dalam folder khusus untuk kemudahan referensi tim.
